DroneEar

AI Acoustic Drone Pattern Identifier

앱 소개

DroneEar는 스마트폰 마이크와 블루투스를 이용하여 주변 드론의 음향 패턴을 실시간으로 분석하고, 유사한 드론 타입을 식별하는 온디바이스 AI 앱입니다. 모든 오디오 처리는 기기 내에서만 이루어지며, 외부 서버로 데이터가 전송되지 않습니다.

주요 기능

  • AI 기반 실시간 드론 음향 패턴 분석
  • BLE Remote ID (ASTM F3411) 스캔
  • 실내/실외 환경 자동 감지 및 감도 조절
  • 나침반 기반 음원 방향 추정
  • 15개 언어 음성 알림 지원
  • 100% 온디바이스 처리 (프라이버시 보호)

자주 묻는 질문

어떤 권한이 필요한가요?
마이크(음향 분석), 위치(실내/실외 판별), 블루투스(Remote ID 스캔), 모션 센서(방향 추정)에 대한 권한이 필요합니다. 모든 데이터는 기기 내에서만 처리됩니다.
녹음이 저장되나요?
아니요. DroneEar는 오디오를 녹음하거나 저장하지 않습니다. 마이크 입력은 실시간으로 분석된 후 즉시 폐기됩니다.
인터넷 연결이 필요한가요?
아니요. 모든 AI 분석은 기기 내에서 수행되므로 오프라인 환경에서도 사용할 수 있습니다.
식별 결과는 정확한가요?
DroneEar는 음향 패턴을 기반으로 유사한 드론 타입을 추정하는 참고용 도구입니다. 환경 소음, 거리, 기기 성능에 따라 결과가 달라질 수 있으며, 100% 정확한 식별을 보장하지 않습니다.
배터리 소모가 큰가요?
백그라운드 모니터링 시 일정 수준의 배터리가 소모됩니다. 앱 내에서 배터리 잔량을 모니터링하며, 낮은 배터리 상태에서는 알림을 제공합니다.
